üìñ Description
================
This project is a basic implementation of a static website using HTML, CSS, and JavaScript. The website has multiple pages, including Home, About, Events, Movies, and Submission. Each page has a unique design and layout, showcasing the use of CSS for styling and HTML for structure.

The website is fully responsive, meaning it can be accessed and viewed on various devices, including desktops, laptops, tablets, and mobile phones. The use of CSS media queries ensures that the website layout adapts to different screen sizes and orientations.

The project includes various features, such as a navigation menu, footer section, and interactive elements like buttons and links. The website is designed to be user-friendly and easy to navigate, providing a seamless experience for visitors.

üìñ More About the Project
-------------------------

The project is built using a simple and modular approach, with each page having its own HTML, CSS, and JavaScript files. This makes it easy to maintain and update the website, as well as add new features and pages.

The website uses a consistent design language throughout, with a focus on simplicity, clarity, and readability. The use of whitespace, typography, and color schemes ensures that the website is visually appealing and easy to use.

üìñ Project Goals and Objectives

The main goal of this project is to create a basic static website that demonstrates the use of HTML, CSS, and JavaScript. The project aims to showcase the following objectives:

* Implement interactive elements like buttons and links
* Ensure cross-browser compatibility and accessibility

The project has the following structure:
